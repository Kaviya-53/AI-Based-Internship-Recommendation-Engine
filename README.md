## 📌 Project Overview

The **AI-Based Internship Recommendation Engine** is a web application that recommends suitable internships to students based on their skills, uploaded resume, and preferred district in Tamil Nadu.

The system uses **Artificial Intelligence (TF-IDF and Cosine Similarity)** to match student skills with internship requirements and provides the best recommendations.

- 👨‍🎓 Student Registration & Login

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Kaviya-53/AI-Based-Internship-Recommendation-Engine.git
```

### Open Project

```bash
cd AI-Based-Internship-Recommendation-Engine
```

### Install Packages

```bash
pip install -r requirements.txt
```

### Run

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
